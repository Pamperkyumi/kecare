# 快速开始

按照以下步骤，几分钟内即可在本地运行 Kecare 博客。

## 前置要求

| 要求 | 说明 | 验证命令 |
| --- | --- | --- |
| Node.js | 安装依赖、启动开发服务器 | `node -v` |
| Git | 克隆主题仓库| `git --version` |

## 安装 CLI

```bash
npm create kecare@beta
```

验证安装：

```bash
kecare -version
```

## 获取主题

```bash
git clone https://github.com/Pamperkyumi/Kecare-theme.git
cd Kecare-theme
```

## 安装依赖

::: tabs

@tab bun

```bash
bun install
```

@tab npm 

```bash
npm install 
```
@tab pnpm

```bash
pnpm install 
```

:::

## 运行

### 生成站点

```bash
kecare gen .
```

### 启动开发服务器

::: tabs

@tab bun

```bash
bun run dev 
```

@tab npm 

```bash
npm run dev
```

@tab pnpm

```bash
pnpm run dev
```

:::

访问 http://localhost:3000 查看站点。

## 更新

```bash
npm update kecare@beta
```
