# 国际化处理

Kecare 支持 AI 自动翻译，让文章触达多语言读者。

## 启用翻译

在文章 Front Matter 中添加 `translate` 字段：

```yaml
---
title: 我的第一篇文章
translate: ['zh-CN', 'en-US', 'ja-JP']
---
```

> 第一个语言代码必须是文章的**原始语言**。

## 配置翻译服务

在 `.kecare/kecare.config.ts` 中配置 LLM：

```ts
export async function kecareConfig() {
    return {
        llm: {
            model: 'deepseek-chat',
            apiKey: 'your-api-key',
            apiBaseUrl: 'https://api.deepseek.com/v1',
            prompt: ''  // 可选：额外翻译提示词
        }
    };
}
```

支持任何兼容 OpenAI API 格式的服务。

## 常用语言代码

| 语言 | 代码 |
|------|------|
| 简体中文 | `zh-CN` |
| 英语 | `en-US` |
| 日语 | `ja-JP` |
| 韩语 | `ko-KR` |
| 繁体中文 | `zh-TW` |

语言代码遵循 BCP 47 标准：`语言-地区`（如 `en-GB`、`pt-BR`）。

## 缓存与增量更新

翻译结果缓存在 `.kecare/.tmp` 目录，下次构建时仅翻译修改部分。

清除缓存：

```bash
rm -rf .kecare/.tmp
```
