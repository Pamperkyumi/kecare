# 关于 Kecare

Kecare 是一个静态博客生成器，核心特点是**非侵入式设计**——把内容系统从主题/框架中解耦，让你的博客能力变成"可插拔模块"。

## 核心优势

传统博客方案（如 Hexo）会绑定特定技术栈，更换技术栈就得重来。

Kecare 的解决方案：在项目中新建 `.kecare/` 目录，按约定写好内容与规则，Kecare 根据规则生成博客页面和数据。

**支持任意技术栈**：Vue、React、Svelte、传统 SSR 等，只要能接入生成结果即可。

## 两种使用方式

**方式一：默认主题**

使用官方提供的 Nuxt + Vue 3 主题，开箱即用，适合快速开始写作。

**方式二：自定义主题**

在任何项目中新增 `.kecare/` 目录，自定义生成规则，让 Kecare 输出你项目可消费的产物。

## 设计理念

- 博客能力是附加的，不需要重构现有项目
- 内容与主题解耦，可移植
- 支持任意技术栈，不绑定
