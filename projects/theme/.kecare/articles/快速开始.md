# 快速开始

如果你想快速体验 Kecare 搭建博客，不打算立刻研究生成器原理或二次开发，按照这篇文档操作就能在本地运行起来。

---

## 前置要求

### 必需项

| 要求 | 说明 | 验证命令 |
|------|------|----------|
| Node.js | 用于安装依赖、启动开发服务器 | `node -v` |
| 主题项目 | 负责渲染页面 | - |

### 可选项（推荐）

| 要求 | 说明 |
|------|------|
| Git | 方便克隆主题仓库（没有也可以下载 ZIP 解压） |

---

## 安装 Kecare CLI

### 步骤一：执行安装命令

在任意目录打开终端，执行：

```bash
npm create kecare@beta
```

### 步骤二：验证安装

```bash
kecare -version
```

如果能输出版本信息，说明安装成功。

---

## 获取示例主题

### 方式一：使用 Git 克隆

```bash
git clone https://github.com/Pamperkyumi/Kecare-theme.git
cd Kecare-theme
```

### 方式二：下载 ZIP

1. 访问 [Kecare-theme 仓库](https://github.com/Pamperkyumi/Kecare-theme)
2. 点击 **Code** → **Download ZIP**
3. 解压并进入目录

---

## 安装依赖

### 步骤一：进入主题目录

```bash
cd Kecare-theme
```

### 步骤二：安装项目依赖

```bash
npm install
```

### 步骤三：安装 Kecare

```bash
npm install kecare@beta
```

> 💡 **提示**：你也可以使用 pnpm、yarn 或 bun 来安装依赖。

---

## 运行生成器

### 构建站点

使用 `kecare gen` 命令生成站点：

```bash
# 在主题目录中
kecare gen .

# 或指定主题路径
kecare gen ./Kecare-theme

# 或使用绝对路径
kecare gen /path/to/Kecare-theme
```

### 启动开发服务器

```bash
kecare dev
```

---

## 验证启动成功

### 查看输出

如果一切正常，你会看到类似输出：

```bash
$ nuxt dev
● Nuxt x.x.x ...
➜ Local: http://localhost:3000/
```

### 访问站点

打开浏览器访问：

```
http://localhost:3000/
```

能看到站点首页就说明你已经成功运行起来了！

## 常见问题

### Q: 如何更新 Kecare？

A: 执行以下命令：

```bash
npm update kecare@beta
```

开始你的 Kecare 之旅吧！ 🚀
