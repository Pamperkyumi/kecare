# 写作

在 Kecare 中，文章存放在 `主题目录/.kecare/articles/` 目录下，目前仅支持 Markdown 文件。

## 创建文章

1. 在 `.kecare/articles/` 目录下创建 `.md` 文件
2. 编写内容

> 文件名建议使用小写字母 + 连字符，如 `getting-started.md`。避免频繁更改文件名，否则可能导致旧链接失效。

## 文章结构

文章由 **Front Matter** 和 **正文内容** 组成：

```markdown
---
title: Hello Kecare
date: 2026-01-22 12:00:00
tags:
  - kecare
  - blog
desc: 文章摘要
author: Pamper
menu: test
translate: ['zh-CN', 'en-US', 'ja-JP']
---

# Hello Kecare

正文内容...
```

## 运行生成器

写完后运行生成器即可在浏览器中预览。
